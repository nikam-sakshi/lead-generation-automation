# 🚀 Lead Generation Automation System using Python

## 🎯 Objective
This project automates the process of collecting, cleaning, and organizing lead data using Python. It extracts key fields, handles missing values, generates email formats, and exports the final dataset into an Excel file.

## ⚙️ Features
- Data loading from dataset  
- Data cleaning (duplicates & missing values)  
- Extract required fields (Name, Email, Website, Location)  
- Email format generation  
- Automation trigger using function  
- Excel file export  

## 🧰 Tools Used
- Python  
- Pandas  
- Google Colab  
- OpenPyXL

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Open the notebook:

```
analysis.ipynb
```

## 📂 Output
Final cleaned dataset saved as `final_leads.xlsx`

## 🚀 Outcome
A simple end-to-end automation pipeline for lead generation and data processing.
